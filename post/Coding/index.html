基于Gridea+Coding搭建个人静态博客

2020-08-28 3 min read # 静态网站 # Gridea

因为某些原因，Coding页面改版，域名也很乱，暂时没有探索出更好的解决方案！

Coding

Gitee Pages可以试试~
本教程为帮助大家搭建自己的博客
起因
emm...有灵感就写
一.需要准备的
1.一个思维灵活的大脑
2.一台不卡的计算机
二.开始搭建网站
首先我们下载Gridea （控制你的网站）
Gridea下载链接

是哪个系统就下载那个，我这里用Windows10示例。
注册/登录Coding链接


创建项目

这里选择DevOps项目

选择一个名称，统一名称并启用 README.md 文件初始化项目，公开源代码（公开后，任何人都可以访问代码仓库，请慎重考虑！） 请勾选此部（很重要）

点击完成创建即可

点击>持续部署>静态网站，先实名认证无需拍照输入身份证即可
点击立即发布静态网站
网站名称和创建项目的名称一样
部署来源选择本项目仓库
选择项目：xxx
选择仓库：xxx
网站名称：xxx
触发机制：自动部署，勾选代码更新时自动部署
推送到master是触发构建
点击保存，Coding自动为你创建一个网站
打开访问地址显示404，因为还没有文章推进仓库。点击立即部署
之后输入你的Coding网站域名

仓库名称<返回首页>点击项目>点击你创建的博客项目>点击代码仓库>点击设置>点击访问设置>当前代码仓库已开源，开源访问地址 :网址https://xxx.coding.net/public/xxx/xxx/git>点击访问,最后一张图是你的仓库名称




分支：master
仓库用户名：点击你的个人账户设置

这里是我画红圈的地方是仓库用户名
邮箱是你注册Coding的邮箱
令牌用户名是个人账户选择：访问令牌



注意：去句号
新建令牌，什么名称都可以

勾选project:depot
创建令牌，输入密码

CANME无需填写，如果你有域名可以填写
Gridea单击保存，检测远程连接即可
之后就可以尽情发挥创意了。
Gridea有很多主题（除了默认主题还有很多）

去Gridea下载一款心愿的主题~

点击主题市场，或点击我的博客

我用的主题是Fog
下载一下~
点击这里


把主题文件解压
拉动到Gridea>thems>主题 即可



等待几次404之后即可看到界面
若依然无效建议用Shift+F5(强制刷新)
帮助(或许对你有用)
我用的主题是Fog
有很多自定义配置 等待你去探索~
