PicGo+Gitee搭建个人图床

2020-11-21 2 min read # 私人云盘 # Gitee # PicGO

搭建私人图床对于博主来说就是减少站内图片加载，减少服务器压力
首先下载一下PicGO，对应的系统下载对应的版本
PicGo下载
现支持SM.MS图床 腾讯云COS Github 七牛 Imgur 阿里云OSS 又拍云 Gitee等
我们搭建Gitee，没有账号请自行注册登陆地址
新建仓库


此处是否开源点击公开，仓库名称任意
并使用Readme文件初始化这个仓库，单击创建即可
创建完成点击个人头像，并点击个人主页

并点击个人设置

选择私人令牌

生成新令牌，描述随便填


全选即可，点击提交，并输入Gitee账号密码

一定要保存新令牌，复制一下，建议别关闭页面

打开PicGO
在插件设置里搜索Gitee,并安装Gitee2.0.3插件，耐心等待一会


owner是你的用户名，不是你的中文用户名，在Gitee的个人主页里
@xxxx，别把@添加上哦
repo:这是你创建的仓库名

path 无需填写
token 是你的令牌，复制粘贴即可
message 无需填写
点击确定，并设置为默认图床
你也可以点击PicGO设置将无用的选项删除
最后点击上传

选择你的链接格式即可
最后点击相册，即可复制
不懂可以提问~
